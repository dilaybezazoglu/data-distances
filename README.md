Bu challenge için her şeyi VS Code içinde yapın.

Birkaç ipucu:

* VS Code’da bir notebook’un ilk hücresini çalıştırdığınızda, ekranın üst kısmından kernel seçmeniz gerekir.

  * Önce "Select Python Environment" seçin.

* Terminal açmak için üst menüde `Terminal` → `New Terminal` yolunu izleyin.

Tüm terminal komutlarını (git, pytest, make, ...) notebook içinden değil, terminalden çalıştırın.
